# README
## サービス概要ーMochibe
Mochibe は、
毎日のモチベーションの波を「3 WHY チャット形式」で深掘りし、感情の原因を客観視できるセルフコーチングツール です。

1日の初めか終わりに、AI との簡単なチャットを通じて

・今日はモチベが上がった／下がった理由
・それが本質的に何から来ているのか
・明日をどう過ごせばいいのか

を整理できます。

また、週・月・年単位のグラフでモチベーションの変化を振り返り、
行動・生活習慣・感情パターンを可視化して、自分自身の改善につなげます。

## このサービスへの思い・作りたい理由
私は昔から、モチベーションに強く左右されるタイプでした。

やる気がある日は一気に集中できる。逆に、やる気が落ちると思うように進めれなくなる

小さなことでも気分が乱れるとペースが一気に崩れる

そんな自分に、何度も振り回されてきました。

一方で、
「どんな日でも一定のペースで、安定したメンタルで前に進める人」がいます。
そういう人を、私は心から尊敬しています。

そして思いました。

「モチベーションに振り回されず、一定のペースで生きられる自分になりたい」
「そのためには、毎日の感情の揺れを“見える化”し、自分と対話する仕組みが必要だ」

Mochibe は、そんな私自身の課題から生まれた
自己対話 × 感情可視化 × 習慣改善 のためのツールです。

自分のために作り始めたものですが、
同じように悩む誰かの役に立てたら、これほど嬉しいことはありません。

## ユーザー層について
・日々のモチベーションに振り回されがちなユーザー
・自分自身の感情や行動パターンをうまく理解できていないユーザー
・定期的に自己を振り返り、改善のヒントを得たいユーザー
・継続が苦手で、「コツコツできる自分」になりたいと願うユーザー

## サービスの利用イメージ
ユーザーは、1日の始まり　または 終わりにこのサービスを使って、自分のモチベーションを短いチャット形式で整理します。

朝に利用する場合
・今日のモチベーションの状態を　3WHY　で深堀し確認
・昨日の気分の揺れを簡単に振り返る
・今日をより良く過ごすための行動指針を明確にできる

夜に利用する場合
・今日のモチベーションの上下を客観的に振り返る
・その日の出来事を 3WHY で深掘りし原因を整理
・明日への前向きな意気込みや改善ポイントを言語化できる

こうした振り返りデータは、週・月・年ごとのグラフとして可視化され、
自分の傾向（落ちやすい曜日・上がりやすい要因・改善サイクルなど）がわかるようになります。

結果として、
「モチベーションに振り回される側」から
「モチベーションを理解してコントロールできる側」へ、
ユーザーの変化をサポートするサービスです。

## ユーザーの獲得について
想定ユーザー層に対して、以下のような方法でサービスを届けることを考えています。

SNS発信（X・Instagram）：日々のモチベや悩みを抱えやすい層へ自然にリーチ。

学習コミュニティ（RUNTEQなど）でのシェア：自己改善に関心のある人に届きやすい。

友人・知人への紹介：実際の使用感を伝え、信頼性のあるユーザー獲得を。

## サービスの差別化ポイント・推しポイント
このサービス Mochibe の最大の特徴は、一般的な日記アプリや習慣管理ツールとは違い、
“モチベーションの原因を言語化して理解すること” に特化している点 です。

その日の気分を「なんとなく」ではなく、
3WHY形式で深掘りして“原因”を特定できる のが最大の強みです。
自分の傾向を見える化することで、
次の日の行動が変わる実感を得られる 点が他サービスとの大きな違いです。
